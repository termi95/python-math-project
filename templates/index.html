Punkty przecięcia odcinków

Punkty przecięcia odcinków to wspólne punkty, w których dwa odcinki na płaszczyźnie przecinają się. Ich współrzędne można obliczyć, rozwiązując równania prostych opisujących te odcinki oraz upewniając się, że znalezione punkty leżą wewnątrz obu odcinków.

def orientation(p: Point, q: Point, r: Point):
 val = (q.y - p.y) * (r.x - q.x) - (q.x - p.x) * (r.y - q.y)

 if val == 0:
  return 0
 return 1 if val > 0 else 2

def on_segment(p: Point, q: Point, r: Point):
 return (
   q.x <= max(p.x, r.x)
   and q.x >= min(p.x, r.x)
  and q.y <= max(p.y, r.y)
  and q.y >= min(p.y, r.y)
)
